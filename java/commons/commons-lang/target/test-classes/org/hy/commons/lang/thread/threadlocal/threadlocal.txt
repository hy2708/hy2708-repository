如何理解同步与ThreadLocal
===================================
在多线程的环境下，对于类的成员变量的操作
若是，通过接口实现的，多线程
那么，类成员变量，是被多个线程共享的，
若是被共享的变量，资源内部没有属性变量（是无状态的），那么可以不考虑，多线程的线程安全问题了
若是不希望该变量，被共享，那么可以使用Threadlocal，即每个线程，各自拥有变量的副本
